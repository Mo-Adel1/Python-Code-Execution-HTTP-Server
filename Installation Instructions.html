2- Prerequisites

  • This project requires Python 3.6 or later. To check your Python version, run in Windows cmd:
    python --version

    or in Linux terminal:

    python3 --version

    If you need to install Python, download the latest version for your operating system from

    Python Downloads.

3- Installation and Running Instructions

After extracting the compressed file:

For Windows

  1. Navigate to the extracted project directory:
    Open the Command Prompt and change to the directory where you extracted the project files:
    cd \path\to\extracted\project
  1. Create a Virtual Environment (make sure you are in the root folder):
    python -m venv venv
  1. Activate the Virtual Environment:
    .\venv\Scripts\activate
  1. Install Required Dependencies:
    pip install -r requirements.txt
  1. Run the Server:
    python main.py

For Linux

  1. Navigate to the extracted project directory:
    Open a terminal and change to the directory where you extracted the project files:
    cd /path/to/extracted/project
  1. Create a Virtual Environment (make sure you are in the root folder):
    python3 -m venv venv
  1. Activate the Virtual Environment:
    source venv/bin/activate
  1. Install Required Dependencies:
    pip install -r requirements.txt
  1. Run the Server:
    python3 main.py

4- Testing the Server with Postman

For easy testing, a Postman Collection has been provided that contains basic examples and unit tests for the server.

Postman Collection: HTTP-Server.postman_collection.json

This collection includes pre-configured requests and tests to help you quickly test the server's functionality. The collection is located in the root directory of the project.

How to Import the Collection in Postman:

  1. Open Postman.
  1. Click on the Import button (located in the top-left corner).
  1. In the import window, select the File tab.
  1. Click on Choose Files and select the HTTP-Server.postman_collection.json file from the root directory of your project.
  1. Click Open, and the collection will be imported into Postman.

How to Run the Collection:

Once the collection is imported, you can run it by following these steps:

  1. Go to the Collections tab in Postman.
  1. Select the HTTP-Server collection.
  1. Click on the Run button (located at the top-right of the collection view).
  1. Then click Start Run.

Postman Tests

The collection includes unit tests in JavaScript that validate the responses from the server, ensuring that the server is working as expected. If any test fails, Postman will provide feedback so you can troubleshoot.
